How to Create Splash Screens:

  1. Upload your logo using the "Logo Image" field above
  2. Adjust the logo size, background color, and text as needed
  3. Select each device size from the dropdown
  4. Click "Download Splash Screen" for each size
  5. Files will auto-download with correct names (e.g., splash-iphone-16-pro-max.png)
  6. Place all files in frontend/public/
  7. Deploy and test on your iPhone!
